Import Demo content, widgets and theme settings for wphobby themes

== Description ==

WPHobby Demo Import includes pre-made templates and sections and design your website to stand out from the crowd.
You can check our website demo on <a href="https://wphobby.com/demo"> Hasium Demo</a> which is build base on our premium theme <a href="https://wphobby.com/"> Hasium Pro Version</a>

== Installation ==

1. Unzip the downloaded zip file.
2. Upload the plugin folder into the `wp-content/plugins/` directory of your WordPress site.
3. Activate `WPHobby Demo Import` from Plugins page.
